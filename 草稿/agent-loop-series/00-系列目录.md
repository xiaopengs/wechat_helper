## 术语说明(系列全文统一)

- **Goal Loop / 目标循环**:Anthropic 文档里的正式说法。本系列用它指代"以目标为起点,自规划-执行-测试-验证,直到目标达成的循环",包括 Codex /goal、Claude /goal、ghuntley 的 Ralph loop、ralphable 的产品化形态。**"Ralph Loop"是社区称呼(源于 Geoffrey Huntley 的 Ralph Wiggum 梗),本系列在跟 ghuntley/ralphable 上下文里用它,其他场景用"Goal Loop"**。
- **Subagent / 子智能体**:泛指"在主 agent 上下文中被委派、有独立上下文窗口、返回结果给主 agent 的工作单元"。各厂术语:Cursor *Subagent*、Anthropic *Subagent*、Codex *sub-agent*。
- **Orchestrator-Subagent / Orchestrator-workers**:Anthropic 5 种模式里的同一种,主 agent 动态拆任务给 worker。本系列统一用 *Orchestrator-Subagent*。
- **Multi-agent orchestration**:Anthropic 给的现象级搜索词,2026-05 同比 +376%(Futurum Research / FindSkill.ai 数据)。
