# 公众号账号诊断框架

## 一、账号画像评分体系（100分）

### 1.1 定位清晰度（25分）

| 评分 | 标准 |
|------|------|
| 21-25 | **极清晰**：一句话就能说清"谁、为谁、提供什么价值"；头像+简介+名称三位一体，形成完整认知 |
| 16-20 | **清晰**：定位明确，但三者（头像/简介/名称）之间略有错位，读者需要3秒以上理解 |
| 11-15 | **模糊**：有方向但不聚焦，简介像口号不说具体价值，读者看完不知道能获得什么 |
| 6-10 | **混乱**：内容主题跨度大，从育儿到AI什么都写，没有明确赛道标签 |
| 0-5 | **无定位**：名称像个人昵称，简介空白或无意义，内容完全随性 |

### 1.2 风格一致性（25分）

| 评分 | 标准 |
|------|------|
| 21-25 | **高度一致**：标题风格、封面设计、段落节奏、语言调性每篇高度统一，形成强识别性 |
| 16-20 | **基本一致**：大部分文章风格统一，偶有偏离；读者能认出是这个账号的内容 |
| 11-15 | **略有波动**：风格有变化但大体在同一方向；时而专业时而通俗，读者略感困惑 |
| 6-10 | **不一致**：今天像新华社，明天像朋友圈；封面图从极简到花哨跳跃极大 |
| 0-5 | **完全混乱**：每篇风格都不一样，像多个账号共用一个公众号 |

### 1.3 内容差异化（25分）

| 评分 | 标准 |
|------|------|
| 21-25 | **极强差异化**：同赛道找不到第二个；有独特的分析框架、表达方式、或数据来源 |
| 16-20 | **有差异化**：有自己的特色视角或方法论，但不是独家；读者能说出"这个号的特点是..." |
| 11-15 | **略有差异**：比普通账号好一点，但特色不鲜明；换个名字读者可能认不出来 |
| 6-10 | **同质化严重**：内容都是大路货，和同赛道10个以上账号内容高度重合 |
| 0-5 | **洗稿搬运**：大部分内容来自其他账号的整合、洗稿或直接搬运 |

### 1.4 更新规律性（25分）

| 评分 | 标准 | 对应频率 |
|------|------|---------|
| 21-25 | **极规律**：固定周几几点更新，误差不超过1小时，连续3个月无断更 | 周更≥2次 |
| 16-20 | **规律**：更新频率稳定，偶尔断更但不超过3天，读者有预期 | 周更≥1次 |
| 11-15 | **基本规律**：有更新但不固定时间，偶尔周更2次偶尔2周不更，读者无法预期 | 月更≥4次 |
| 6-10 | **不规律**：想起才更，间隔从1天到1个月不等，像个人树洞 | 月更≥1次 |
| 0-5 | **僵尸号**：月更<1次，或已断更超过2个月 | 月更<1次 |

## 二、主题与调性诊断维度

### 2.1 主题集中度诊断

| 状态 | 判断标准 | 建议 |
|------|---------|------|
| **过度聚焦** | 单一主题占比 > 90% | 风险：内容枯竭快，读者审美疲劳。建议：适度拓展周边1-2个相关主题 |
| **健康聚焦** | TOP3主题占比 70-90% | 状态健康，保持即可 |
| **略有分散** | TOP3主题占比 50-70% | 建议：收窄到2个核心方向，砍掉表现最差的主题 |
| **严重分散** | TOP3主题占比 < 50% | 紧急：必须做取舍，否则账号永远长不大 |

### 2.2 调性清晰度诊断

**语言风格矩阵**：
```
专业↑
  | 学术型    分析型
  |
  | 教程型    资讯型
通俗→
```

**价值观倾向**：
- **技术乐观派**：相信AI改变世界，强调效率提升，案例多为成功实践
- **审慎批判派**：关注风险、失业、伦理问题，对技术 hype 保持警惕
- **实用主义派**：不站队，只讲"怎么用""怎么赚钱""怎么提效"
- **价值中立派**：客观呈现事实，很少表达主观倾向

**目标人群分层**：
- **入门级**：大量解释基本概念，适合小白
- **进阶级**：假设读者有基础，讲方法讲案例
- **专家级**：讲底层逻辑，做深度分析，面向行业从业者

## 三、内容风格诊断要点

### 3.1 标题风格分类

| 类型 | 特征 | 适用场景 | 点击预期 |
|------|------|---------|---------|
| **数字型** | 含具体数字，如"7个方法""3000字长文" | 干货、清单、长文 | 高 |
| **悬念型** | 留问号或悬念，如"没想到"、"你还不知道" | 热点、揭秘、观点 | 中高 |
| **痛点型** | 直击读者痛点，如"别再浪费钱了" | 避坑、经验分享 | 中 |
| **观点型** | 鲜明表达立场，如"我反对AI替代论" | 争议话题、深度文 | 中低 |
| **名人型** | 借势名人/公司，如"马斯克又出手了" | 新闻、热点 | 极高但风险高 |
| **提问型** | 向读者提问，如"你被AI取代了吗" | 互动、调查 | 低 |

### 3.2 文章结构类型

| 类型 | 特征 | 适用内容 | 阅读体验 |
|------|------|---------|---------|
| **清单体** | 1/2/3/4/5点罗列 | 干货、经验、方法 | 好，易读 |
| **故事体** | 以故事开头/贯穿全文 | 案例、人物、创业 | 极好，共情强 |
| **深度分析** | 提出问题-分析-结论 | 行业观察、趋势 | 中，需耐心 |
| **资讯汇总** | 多条新闻+短评 | 早报、周报 | 好，信息密度高 |
| **教程指南** | 步骤+截图+代码 | 技术、工具 | 中，实用但枯燥 |
| **访谈对话** | 问答形式 | 人物、经验分享 | 中，因人而异 |

### 3.3 人味指数判断

**高AI风险信号**（出现3个以上需警惕）：
- 段落长度高度一致（都是3-5行）
- 过度使用"首先/其次/再次/最后"结构
- 标准的总-分-总结构，没有例外
- 缺少个人经历、具体案例、真实细节
- 情感表达中性，看不到作者的喜怒哀乐
- 过度使用emoji但没有实质情绪

**高人气信号**（出现2个以上加分）：
- 有作者的个人故事或经历分享
- 有具体的人名、公司名、产品名
- 有数字、有细节、有场景感
- 有鲜明的观点倾向（哪怕有争议）
- 有口语化表达、自嘲、幽默

## 四、诊断输出规范

所有诊断结论必须遵循：
1. **有数据**：每个结论后面标注依据（如"主题分散：TOP3主题占比仅42%"）
2. **有对比**：给出同赛道基准值（如"行业平均在看率1.2%，本账号0.7%"）
3. **有案例**：举账号内具体的正面/反面例子（如"5月15日《XXX》这篇标题就是典型数字型"）
4. **有建议**：每个问题对应1-2个可执行的优化动作
