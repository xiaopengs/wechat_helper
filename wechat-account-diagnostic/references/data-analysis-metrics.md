# 公众号数据分析指标体系

## 一、核心指标定义与计算

### 1.1 基础流量指标

| 指标 | 公式 | 说明 |
|------|------|------|
| **平均阅读量** | 总阅读数 / 文章数 | 账号整体影响力的核心指标 |
| **中位数阅读** | 阅读量排序后的中间值 | 比平均值更能反映真实水平，排除极端值 |
| **阅读波动系数** | 阅读量标准差 / 平均阅读 | <0.3稳定，0.3-0.5正常，>0.5波动大 |
| **最高阅读** | 单篇最高阅读 | 账号潜力上限 |
| **最低阅读** | 单篇最低阅读 | 账号底线下限 |

### 1.2 互动转化指标

| 指标 | 公式 | 优秀阈值 | 一般阈值 | 需改进 |
|------|------|---------|---------|--------|
| **在看率** | 在看数 / 阅读数 | > 1.5% | 0.8-1.5% | < 0.8% |
| **点赞率** | 点赞数 / 阅读数 | > 0.8% | 0.5-0.8% | < 0.5% |
| **评论率** | 评论数 / 阅读数 | > 0.3% | 0.15-0.3% | < 0.15% |
| **互动率** | (在看+点赞+评论) / 阅读 | > 2.5% | 1.5-2.5% | < 1.5% |

> **说明**：在看率是最重要的互动指标，因为"在看"会出现在朋友的看一看中，带来二次传播。评论率高说明内容引发思考或争议，是深度互动的标志。

### 1.3 增长相关指标（需要历史对比）

| 指标 | 计算方式 | 健康标准 |
|------|---------|---------|
| **阅读环比增速** | (本月平均 - 上月平均) / 上月平均 | > 10% 增长，±5% 平稳，< -10% 下滑 |
| **新关注数** | 后台数据 / 第三方估算 | 与阅读量正相关，转化率 1-3% 正常 |
| **取关率** | 取关数 / 新增关注数 | < 10% 健康，> 20% 内容有问题 |

## 二、时间维度分析

### 2.1 发布时间分析

**最佳发布时间窗口（行业大数据）**：

| 时段 | 时间 | 打开率 | 适用内容类型 |
|------|------|--------|------------|
| **早高峰** | 7:30-9:00 | 高 | 资讯、早报、短内容 |
| **午休** | 12:00-13:30 | 中高 | 轻松内容、故事、干货 |
| **下班前** | 17:00-18:30 | 中 | 新闻、热点、短文章 |
| **晚高峰** | 20:30-22:30 | 最高 | 深度长文、分析、情感类 |
| **深夜** | 23:00+ | 低 | 不建议 |

**分析方法**：
1. 将账号历史文章按发布时间分组（1小时间隔）
2. 计算每组的平均阅读、互动数据
3. 与行业基准对比，找出账号的"黄金时间"
4. 检查：是否总是在数据表现最差的时间发布？

### 2.2 周几表现分析

| 周几 | 行业平均打开率 | 特点 |
|------|---------------|------|
| 周一 | 中低 | 刚上班，注意力在工作 |
| 周二 | 高 | 工作进入状态，有间隙阅读 |
| 周三 | 最高 | 周中，阅读习惯稳定 |
| 周四 | 中高 | 接近周末，注意力开始分散 |
| 周五 | 中 | 心思在周末，适合轻松内容 |
| 周六 | 低 | 户外活动多 |
| 周日 | 中 | 晚上阅读率回升 |

### 2.3 趋势分析方法

**移动平均法**：
- 计算最近7篇、14篇、21篇文章的移动平均阅读
- 观察三条线的走势：向上/向下/平稳
- 如果7日平均持续低于21日平均 → 下滑趋势

**拐点识别**：
- 连续3篇阅读低于平均的70% → 可能出现拐点
- 突然出现1篇阅读远超平均（>200%）→ 爆款，分析原因复制
- 突然出现1篇阅读远低于平均（<30%）→ 踩坑，分析原因避免

## 三、内容维度关联分析

### 3.1 主题-数据关联矩阵

| 主题 | 文章数 | 平均阅读 | 平均在看率 | 表现评级 | 建议 |
|------|--------|---------|-----------|---------|------|
| 主题A | 8 | 5200 | 1.8% | ★★★★★ | 加大投入 |
| 主题B | 5 | 3800 | 1.2% | ★★★ | 保持观察 |
| 主题C | 6 | 2100 | 0.6% | ★ | 减少或停更 |

**分析要点**：
1. 不能只看阅读量，还要看互动率（在看率更重要）
2. 高阅读低互动 → 标题党，内容质量跟不上
3. 低阅读高互动 → 内容好但标题/封面差，或受众太窄
4. 高阅读高互动 → 真正的爆款，分析可复制要素

### 3.2 标题类型-点击率分析

**点击率估算方法**（无精准数据时）：
- 假设公众号平均打开率为 3-5%（行业基准）
- 阅读量 / 预估粉丝数 = 估算打开率
- 不同标题类型的打开率相对对比更有意义

**标题类型效果对比表（示例）**：

| 标题类型 | 占比 | 平均阅读 | 相对点击率 | 结论 |
|---------|------|---------|-----------|------|
| 数字型 | 35% | 4800 | 120% | 表现最好，多用 |
| 悬念型 | 25% | 4500 | 113% | 表现不错 |
| 观点型 | 20% | 3800 | 95% | 一般 |
| 提问型 | 10% | 3200 | 80% | 少用 |
| 其他 | 10% | 3500 | 88% | - |

### 3.3 文章长度-完读率分析

**完读率估算**：
- 无精准数据时，用"在看率"代理完读率（读者需要读完才会点在看）
- 不同字数区间的在看率对比

**字数区间效果（科技/商业类）**：

| 字数区间 | 平均阅读 | 平均在看率 | 结论 |
|---------|---------|-----------|------|
| < 1000字 | 4200 | 1.0% | 阅读高但互动低，看完但没共鸣 |
| 1000-2000字 | 4500 | 1.3% | 最佳区间，阅读和互动平衡 |
| 2000-3000字 | 4000 | 1.5% | 阅读略降但互动高，深度读者 |
| 3000-5000字 | 3200 | 1.8% | 阅读降但互动升，筛选真粉丝 |
| > 5000字 | 2500 | 2.0% | 极致筛选，互动最高但受众少 |

> **策略启示**：
> - 涨粉用 1000-2000 字（平衡阅读和互动）
> - 留粉用 2000-3000 字（建立深度连接）
> - 树立专业形象用 3000+ 字（但不要每篇都写）

## 四、投放数据分析

### 4.1 投放效果核心指标

| 指标 | 公式 | 说明 |
|------|------|------|
| **CPM** | 投放金额 / (曝光量 / 1000) | 千次曝光成本，越低越好 |
| **CPC** | 投放金额 / 点击量 | 单次点击成本 |
| **CTR** | 点击量 / 曝光量 | 点击率，>2% 优秀 |
| **关注成本** | 投放金额 / 新增关注数 | 获客成本，越低越好 |
| **ROI** | 新增关注带来的长期价值 / 投放金额 | 最终效果评估 |

### 4.2 投放-内容匹配分析

**转化漏斗分析**：
```
曝光 → 点击（看素材）→ 阅读（看内容）→ 在看/点赞（看质量）→ 关注（看价值）
  ↓        ↓              ↓                 ↓
素材问题  标题/封面问题   内容质量问题      账号定位问题
```

**常见问题诊断**：
- CTR 低但关注率高 → 素材差，内容好 → 优化投放素材
- CTR 高但关注率低 → 素材好但内容/定位不匹配 → 优化落地文章或账号介绍
- 整体都低 → 投放人群、素材、内容都可能有问题 → 系统性优化

## 五、异常数据识别

### 5.1 数据异常信号

| 异常现象 | 可能原因 | 验证方法 |
|---------|---------|---------|
| 突然某篇阅读暴涨10倍+ | 1. 爆款自然传播<br>2. 被某个大号转载<br>3. 上了推荐/看一看<br>4. 刷量 | 看互动率是否同步增长；刷量通常互动率极低 |
| 连续3篇阅读骤降50%+ | 1. 被限流/降权<br>2. 标题/内容风格突变<br>3. 粉丝大量取关 | 检查后台通知；对比之前的标题风格 |
| 在看率突然>5% | 1. 内容引发强烈共鸣<br>2. 有组织的在看刷量 | 看评论区是否有真实讨论 |
| 评论全是无意义的"好文""学习了" | 大概率刷评论 | 看评论者头像和昵称是否真实 |

### 5.2 数据健康检查清单

每篇文章发布后24小时检查：
- [ ] 阅读量是否在预期区间（平均的±50%）
- [ ] 在看率是否 > 1%
- [ ] 是否有真实评论（不是机器人）
- [ ] 与同主题历史文章对比是否正常
- [ ] 有无明显异常数据（过高/过低）
