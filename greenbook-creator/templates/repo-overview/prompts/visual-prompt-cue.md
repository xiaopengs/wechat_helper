# AI 生图视觉 prompt 速记（如果换成 gen_media.sh / gpt-image-2 出图）

当前模板用 Pillow + 浅色信息图风格（`default`），但如果某个开源项目更适合走 AI 出图，下面的 prompt 结构可以直接喂给 `gen_media.sh --prompt ...`：

```
1080×1440 小绿书{第几页} ·{角色类别} ·{项目名}、{标签}
构图:{视觉主体} 在中央，{附属元素} 在四象限，主色 {HEX1} 渐变到 {HEX2}，
中文 {主标}、{副标}，可读、配角图元素 {icon_family}。
负空间：底部留白 ≥ 20%，左右各留 60px 安全区。
风格：浅色知识漫画 / 信息图，memphis 图标，圆角卡片；
禁用：shadow blur 、宫格分割线、3D 渲染感、AI logo。
```

按页填法：

- **P1 封面**：`{视觉主体}` = 中央"百科 / 工程"图标；`{附属元素}` = 项目目录 + Git + CI + 入口 4 类
- **P2 痛点**：`{视觉主体}` = 3 张痛点小卡 + 1 条流向箭头
- **P3 优势**：`{视觉主体}` = 4 阶段进度条（横排 4 个圆 + 横线）+ 4 优势卡
- **P4 双模式**：`{视觉主体}` = 左右对称双卡（代码 vs 人脑 icon）
- **P5 竞品**：`{视觉主体}` = 3-4 张横长卡（4 条目按"给谁写"选型）
- **P6 上手**：`{视觉主体}` = 暗色 Terminal + 3 张白底建议卡 + 1 条红色风险条

参数：`--model gpt-image-2 --size 1024x1536 --quality high --style default`
（`default` 对应浅色信息图；其它风格选 `tech-dark` / `orange-parenting` 等）
