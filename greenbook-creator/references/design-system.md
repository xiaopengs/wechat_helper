# 小绿书设计系统详解

## 一、设计原则

### 1.1 信息层级

每张卡片的信息层级（从强到弱）：
1. **核心数字/关键词**（最大字号、最醒目颜色）
2. **标题/结论句**（中等字号、加粗）
3. **补充说明**（最小字号、灰色或浅色）
4. **装饰/背景**（不干扰阅读即可）

### 1.2 视觉动线

- 竖屏 3:4 → 从上到下的阅读动线
- 核心信息放在画面中上部（30%-60% 位置）
- 底部留 10% 给品牌/水印/页码

### 1.3 一致性原则

- 同一篇小绿书的所有内页：**同底色、同字体、同排版**
- 变化只通过颜色小变化（如序号变色）实现
- 封面可以稍有不同的设计，但仍保持色系一致

```

内容：[填入具体内容]
要求：留白充足、手机竖屏、1080×1440、无文字错误
```


## 五、工具链建议

### 图片生成
- **gpt-image-2**（通过 TokenRouter）：马卡龙/极简风格生图，适合封面和关键插图
- **Pollinations.ai**：免费备选，简单图形和纯色背景
- **MiniMax image-01**：高质量场景图/产品图

### 图片处理
- 统一裁剪为 1080×1440
- 批量压缩至 ≤ 2MB（JPEG quality 85%）
- 添加统一水印（右下角半透明）

### 发布工具
- 微信公众平台后台 → 图片/文字 → 新建
- 或通过 wechat-draft-publish skill 的 API 接口
