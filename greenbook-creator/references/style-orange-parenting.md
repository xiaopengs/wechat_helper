# 暖橘治愈亲子风（情感共鸣向 · 小绿书 5 页标准结构）

> **核心公式**：60% 情感共鸣 + 30% 可执行方法 + 10% 视觉
>
> 本质是 **「陪伴式说服」**，不是信息堆叠。读者带着情绪来，带着方案走。

---

## 一、风格定位

| 维度 | 取向 |
|---|---|
| **情绪基调** | 治愈 · 共情 · 温柔坚定 · 不是说教 |
| **读者关系** | 平等对话（不是专家俯视）|
| **语言风格** | 第二人称 + 短句 + 行动动词 |
| **适合主题** | 亲子关系、孩子教育、情绪管理、夫妻沟通、自我成长、心理健康 |
| **不适合主题** | 纯技术教程、参数对比、价格清单、攻略合集 |
| **页数规划** | **5 页** 是甜区（封面 + 检测 + 根因 + 4 招 + 升华） |
| **出图引擎** | gpt-image-2（默认走 tokenrouter）/ Gemini 图像系列（需 Gemini 组 key） |

---

## 二、配色体系

### 主色（暖橘渐变）
```
主橘:     #F4A261
深橘:     #E76F51
浅米黄:   #FFE8D6
背景白:   #FFFAF5  (微暖白,不是纯 #FFFFFF)
```

### 辅色
```
柔灰边:   #EAD7C0
深咖文字: #4A3525  (不是纯黑,带温度)
浅灰文字: #8C7B6B
```

### 功能色（点睛用,克制使用）
```
成功绿:   #87B87A  (用于"正反馈"图标)
警示橘红: #E76F51  (用于"踩雷"图标,不要用红色)
```

### 配色禁忌
- ❌ 不要纯黑 `#000000` 当文字色（冷硬）
- ❌ 不要饱和度 > 80% 的颜色（喧闹）
- ❌ 不要超过 3 种饱和色（视觉疲劳）

## 三、字体层级

```
48-60px   主标题(OPPO Sans Medium / MiSans Medium — 不要 Heavy,治愈风不需要力度)
24-32px   副标题(Light)
16-18px   正文(Regular)
14px      注释/页脚
20-24px   大字金句(Light + 字间距 +1px)
```

**核心规则**:标题用 Light/Medium 字重,治愈风靠**留白和温度**,不靠加粗。

---

## 四、插画风格关键词

```
flat illustration                (扁平插画,不是 3D 立体)
minimalist line art              (极简线条)
warm soft orange palette         (暖橘调)
white background                 (留白多)
gentle drop shadow               (柔和投影,不用硬阴影)
rounded soft elements            (圆润元素,无尖角)
no harsh lines                   (线条末端圆润)
warm empathetic mood             (温暖同理心)
professional vector illustration (矢量感,不是水彩手绘)
```

**插画禁忌**:
- ❌ 真人照片风格(成本高且容易"假")
- ❌ 3D 黏土渲染(科技感太强,冲撞治愈风)
- ❌ 表情符号代替插画(显得不专业)

## 五、5 页标准结构(P1-P5)

> 这是**这套风格的页面架构模板**。所有情感共鸣向内容都可以套这个结构,换主题即可。

### P1 · 封面(痛点共鸣 + 一句话定义)

**结构**:
```
大字标题(60px,2 行)
- 第 1 行:场景关键词(让人代入)
- 第 2 行:痛感词(戳情绪)
副标题(24px,1 行)
- "你不是一个人" / "这不是你的错"
底部小字(14px)
- 标签:@某某说 · 育儿观察
```

**示例(育儿场景)**:
> **陪写作业**
> **比加班还累**
>
> 你不是不会教,是你太累了
>
> @某某说 · 育儿观察

**base_prompt 片段**:
```
parenting education card cover,
flat illustration of [场景:妈妈坐在书桌前揉太阳穴],
[情绪:疲惫但坚定],
warm orange palette, white background,
minimalist line art, soft shadow,
1080x1440px, clean typography space at bottom
```

---

### P2 · 自我检测("对号入座" 4 种类型)

**结构**:
```
页头(20px)
- "对号入座,看看你孩子卡在哪个"
4 块横排卡片(每块 1 句描述)
- 每块用 emoji 或简笔插画作为视觉钩子
- 不评分、不判断,只是"识别"
```

**核心心法**:这一页让读者**停下滑动的手指**。识别比建议更稀缺。

---

### P3 · 根因分析(为什么会出现)

**结构**:
```
大字金句(40px)
- 1 句话点破本质
小字解释(16px,3-4 行)
- 不要堆专业术语,用生活类比
底部 1 句方法论(24px)
- "X 不是 Y,关键是 Z"
```

**示例(计时场景)**:
> **计时是为了帮他,**
> **不是为了计时。**
>
> 孩子磨蹭,80% 不是态度问题,
> 是他的**执行功能**还在发育。
> 你越催,他越慌,越慌越慢。

---

### P4 · 4 招破解(直接抄作业)

**结构**:
```
页头(20px)
- "P4 · 4 招破解"
4 块卡片(2x2 或 4 横排)
- 每块:序号 + 一句话方法 + 一个具体动作
- 动作必须能"今晚就试"
```

**示例**:
> **① 按下暂停键**
> "我们都有些情绪,休息 10 分钟再来。"
>
> **② 把「计时」换成「记录」**
> 开个录音,事后一起听。
>
> **③ 用游戏力化解**
> 让他当计时官。
>
> **④ 让慢的自然后果发生**
> 约定 9:30 必须睡,温和坚定执行。

**核心心法**:**动作必须明天就能用**。不要"提升认知""建立边界"这种空话。

---

### P5 · 升华 + 行动引导

**结构**:
```
大字金句(48px,2 行,留白多)
- 上升到"关系"或"长期价值"
小字(16px,3 行)
- 价值锚定 + 行动召唤
互动钩子(14px)
- "评论区告诉我效果"
- "@ 一个你也想@ 的家长"
```

**示例**:
> **养育的本质是关系。**
> **时间观念能补,**
> **信任垮了难修。**
>
> 今晚就试第一招——按下暂停键。
> 评论区告诉我效果,或者 @ 一个你也想 @ 的家长。

**核心心法**:这是**转发动机**。升华 + 召唤 + 互动钩子三件套。

```
flat illustration, soft warm orange color palette (#F4A261 → #FFE8D6),
minimalist line art style, white background with gentle shadow,
parenting education card layout, 1080x1440px,
clean typography space at top and bottom,

```

**参数建议**:

- 尺寸:`1024x1536` 或 `1024x1792`(3:4 竖屏)
- 质量:`medium`(治愈风不需要 high,反而更"塑料")

## 七、文案模板(可直接套)

### 标题公式(情感共鸣向)

1. **场景 + 痛感**:`陪写作业` / `比加班还累`
2. **反问 + 钩子**:`你也在 XX 吗?`
3. **对比 + 选择**:`XX 是为他,还是为你?`
4. **数字 + 共鸣**:`3 个信号,说明你太累了`

### 正文短句公式

1. **你 + 动词 + 不是 + 名词**:`你不是不会教,是你太累了`
2. **不要 + 动词,不要 + 名词,要 + 名词**:`不要催他,要等他自己`
3. **X 不是 Y,关键是 Z**:`计时不是为了计时,关键是关系`

### 升华金句公式(留白多,字数少)

- **本质 + 类比**:`养育的本质是关系`
- **长期 + 反差**:`时间观念能补,信任垮了难修`
- **场景 + 召唤**:`今晚就试第一招`

## 八、反模式(避坑清单)

| ❌ 不要 | ✅ 应该 |
|---|---|
| 一上来讲方法论 | 先让读者代入情绪 |
| 用专业术语 | 用生活类比 |
| 强调"应该" | 强调"今晚就试" |
| 通篇加粗 | 标题用 Light + 留白 |
| 配色饱和度高 | 配色柔和 + 留白多 |
| 真人照片 | 扁平插画 |
| 列表堆砌 | 4 块卡片 + 行动动词 |
| 结尾说"看完点赞" | 升华 + 评论区互动钩子 |

## 九、快速启动(零思考起步)

拿到一个育儿/情感话题,直接套:

1. **P1 封面**:把场景词代入"陪 X 比 Y 还累"
2. **P2 检测**:写 4 种"对号入座"类型
3. **P3 根因**:一句话戳本质 + 3 行解释
4. **P4 4 招**:4 个动作,每个今晚就能试
5. **P5 升华**:上升到大词 + 评论区钩子

**生成脚本**:
```bash
~/.openclaw/workspace/wechat_helper/greenbook-creator/scripts/gen_media.sh generate \
  --prompt "5 张 3:4 育儿教育知识卡片系列, ... (详细 prompt 见 base)" \
  --style orange-parenting \
  --quality medium \
  --size 1024x1536 \
  --out /tmp/orange-card.png
```
