# 微信红包封面完整规格文档

> 📅 最后更新：2026年3月
> 🔗 官方平台：https://cover.weixin.qq.com/

---

## 一、封面主图（必须）

### 基础规格

| 属性 | 要求 |
|------|------|
| **尺寸** | 957 × 1278 像素 |
| **格式** | PNG / JPG / JPEG |
| **文件大小** | ≤ 500KB |
| **颜色模式** | RGB |
| **背景** | 可透明，但建议使用设计背景 |

### 设计注意事项

1. **顶部区域**（约 150px）：会显示发送者头像和昵称，避免放置重要元素
2. **中间区域**：显示金额和祝福语，保持简洁
3. **底部区域**（约 120px）："开"字按钮位置，避开重要内容
4. **边缘安全区**：建议保留 16px 边距

### 静态封面 vs 视频封面

| 类型 | 尺寸 | 格式 | 时长 | 文件大小 |
|------|------|------|------|----------|
| 静态封面 | 957×1278 | PNG/JPG | - | ≤500KB |
| 视频封面 | 957×1278 | MP4 | 1-3秒 | ≤3MB |

### 视频封面额外要求

| 属性 | 要求 |
|------|------|
| 编码格式 | H.264/AVC |
| 帧率 | 建议 30fps |
| 码率 | ≤ 2000 kbit/s |
| 音频 | 无音轨或静音 |

## 二、品牌Logo（可选）

| 属性 | 要求 |
|------|------|
| **尺寸** | ≤ 200 × 200 像素 |
| **格式** | PNG（支持透明） / JPG / JPEG |
| **文件大小** | ≤ 100KB |
| **背景** | 建议透明背景 |

**使用位置**：显示在红包封面详情页

---

## 三、封面故事（可选）

### 图片故事

| 属性 | 要求 |
|------|------|
| **尺寸** | 750 × 1250 像素 |
| **格式** | PNG / JPG |
| **文件大小** | ≤ 300KB/张 |
| **数量** | 最多 5 张 |

### 视频故事

| 属性 | 要求 |
|------|------|
| **格式** | MP4（H.264/AVC 编码） |
| **宽度** | ≥ 720 像素 |
| **宽高比** | 3:5 ~ 16:9 之间 |
| **时长** | ≤ 15 秒 |
| **码率** | ≤ 1600 kbit/s |
| **文件大小** | ≤ 10MB |

## 五、审核规范

### ✅ 设计原则

- 强调装饰性，弱化品牌宣传
- 美观简洁，风格协调统一
- 原创设计或持有完整版权
- 符合社会主义核心价值观

### ❌ 禁止内容

1. 违反法律法规的内容
2. 色情、暴力、赌博相关内容
3. 未经授权的商标或版权素材
4. 二次创作/同人作品（无授权）
5. 过于商业化的广告元素
6. 政治敏感内容
7. 宗教敏感元素

### ⚠️ 特殊审核（延长至1个月）

- 包含外文、少数民族文字（需提供中文翻译）
- 包含宗教、民族、政治性质的图腾、人物、神兽、符号

### 🚫 行业限制

以下行业暂不开放：
- 宗教相关
- 烟草
- 医疗/医疗器械
- 特殊医学用途配方食品
- 农药/兽药
- 保健食品
- 药品

## 六、申请资质

### 企业用户

- 中国大陆境内完成企业认证的微信公众号
- 完成企业认证的视频号用户

### 个人用户（满足任一）

1. 视频号关注人数 ≥ 100人
2. 视频号绑定的微信表情平台账号专辑下载量 ≥ 200次
3. 公众号关注人数 ≥ 100人

---

## 七、定价与分成

### 定制费用

| 项目 | 价格 |
|------|------|
| 单价 | 1元/个 |
| 有效期 | 6个月内发放有效 |
| 最低购买 | 无限制（原100个起） |

### 付费封面功能

**开通条件：**
1. 已开通企业或个人账号
2. 近一年内至少定制并发放过1款封面
3. 近三年内无严重违规记录

**分成规则：**

| 项目 | 比例 |
|------|------|
| 创作者分成 | 80% |
| 平台技术服务费 | 20% |
| 结算周期 | 每月10号结算上月 |

**定价机制：**
- 创作者可自行定价
- 用户通过微信豆兑换
  - 安卓：1元 = 10微信豆
  - iOS：1元 = 7微信豆
